### 部署方法(web)

- ```shell
	//拉取请求
	git@github.com:yanghy949/library-chat-bot-web.git
	```
- ```shell
	//项目安装
	npm install
	```
- ```shell
	//启动项目
	npm run serve
	```
### 完整项目(Docker)
- ```shell
	docker pull registry.cn-chengdu.aliyuncs.com/yanghy949/chatbot:1
  ```
